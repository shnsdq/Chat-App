**Real-Time Chat Application**

* Built a full-stack real-time chat platform using **React.js**, **Node.js**, **Express**, and **MongoDB**, supporting one-to-one conversations.
* Integrated **Socket.IO** to deliver instant messaging, active user presence.
* Used **Zustand** for lightweight and efficient global state management, improving UI responsiveness.
* Designed a clean and responsive interface using **DaisyUI**, enhancing user experience and accessibility across devices.
* Developed modular REST APIs and optimized database queries for faster message retrieval and smooth performance.
